loads.py¶

Load classes for Eurocode-based structural analysis of HP-shell slabs.

Provides an abstract Loads base and the instantiable LoadsEC implementation for EN 1990 / EN 1991 load combinations with the German National Annex (NA/DE). [7]

Author: Elliot Melcer

Abstract base class for load objects.

Defines the interface that all load implementations must satisfy so that structural analysis routines can remain independent of the underlying load model.

class LoadsEC(live_loads, psi_0_values, psi_1_values, psi_2_values, gamma_g=1.35, gamma_q=1.5)[Quellcode]¶

Bases: Loads

Load object for EN 1990 / EN 1991 combinations with German National Annex [7]

Supports uniformly distributed loads applied over all spans. Multiple live load components (e.g. imposed + partitions) can be passed as arrays; the leading variable action always occupies index 0.

combined_line_load_kN_m(slab_construction: SlabConstruction, combination: str = 'FUNDAMENTAL') → float[Quellcode]¶

Compute the combined line load for a given EN 1990 [6] load combination.

Converts the area load of the governing combination to a line load by multiplying by the slab width.

Parameter:

  • slab_construction (SlabConstruction) – Full slab construction object (provides slab width and dead loads).

  • combination (str, optional) – Load combination name. One of "FUNDAMENTAL", "RARE", "FREQUENT", "QUASI_PERMANENT". Default is "FUNDAMENTAL".

Rückgabe:
