# SlabDesignBench

A modular benchmarking framework in Python for the structural optimization of floor slab constructions, addressing the research gap regarding problem-oriented suites which incorporate real-world engineering tasks. As a first slab module, the parametric design tool for the structural optimization of a prestressed carbon concrete HP shell slab construction was translated into Python.

Developed as part of a Master's Thesis at the Technische Universität Berlin.

**Author:** Elliot Melcer  
**Thesis:** *Erstellung einer Benchmarking-Suite für die Strukturoptimierung von HP-Schalen in Python*  
**Institution:** Technische Universität Berlin, 2026  
**API Documentation:** https://elliotmelcer.github.io/382389_ma_slabdesignbench/

## Installation

**Requirements:** Python 3.12+

**Option A: Clone the repository**

```bash
git clone https://github.com/elliotmelcer/382389_ma_slabdesignbench.git
cd 382389_ma_slabdesignbench
```

**Option B: Extract the ZIP archive**

Extract the ZIP and navigate into the folder. The `.venv` is not included and must be created manually (same as Option A).

**Create a virtual environment and install dependencies:**

```bash
python -m venv .venv

# Windows
.venv\Scripts\activate

# Linux / macOS
source .venv/bin/activate

pip install -r requirements.txt
```
